# Piedra, Papel o Tijera 🪨📄✂️

Este es el proyecto del **Foundations** de [The Odin Project](https://www.theodinproject.com/).

[**Ver el proyecto en vivo (Live Preview)**](URL_DE_TU_GITHUB_PAGES_AQUI)

## 📝 Sobre el proyecto

El objetivo de este proyecto fue poner en práctica todos los conceptos fundamentales de JavaScript, pasando de un simple script de consola a una interfaz gráfica web completamente interactiva. El juego enfrenta al usuario contra la computadora, llevando un registro del puntaje hasta que uno de los dos alcance los 5 puntos.

## ✨ Características

* **Interfaz Interactiva:** Botones con efectos hover y transiciones suaves.
* **Lógica de Juego Completa:** La computadora elige su jugada de forma completamente aleatoria.
* **Marcador Dinámico:** El DOM se actualiza en tiempo real mostrando el resultado de cada ronda y el puntaje acumulado.
* **Condición de Victoria:** El juego se detiene automáticamente cuando un jugador llega a 5 puntos.
* **Reinicio Dinámico:** Un botón de "Volver a jugar" se genera automáticamente al finalizar la partida, permitiendo resetear el estado sin recargar la página.

## 🛠️ Tecnologías utilizadas

* **HTML:** Estructura semántica.
* **CSS:** Diseño responsivo, Flexbox y estados de interacción (`:hover`, `:disabled`).
* **JavaScript(Vanilla):** Lógica del juego, manipulación del DOM y manejo de eventos.

## 🧠 Conceptos aplicados

Durante el desarrollo de este proyecto, afiancé mis conocimientos en:
* Uso de variables, funciones y estructuras de control (`if/else`, loops).
* Métodos del objeto global Math (`Math.random()`, `Math.floor()`).
* Selección de elementos del DOM (`querySelector`, `querySelectorAll`).
* Escucha y manejo de eventos (`addEventListener`).
* Creación y destrucción de nodos en el DOM (`createElement`, `appendChild`, `remove`).
* Alcance de variables (Scope) y estado de la aplicación.
